# Soluciones

Aqui viven las soluciones de ejercicios de niveles 1 a 3 (RFC-0001 §17).

Cada solucion debe tener el mismo estandar que el codigo principal: nombres
claros, pruebas cuando aplique, explicacion de complejidad y ninguna solucion
"rapida" que enseñe malos habitos.

Los desafios de nivel 4 no tienen una unica solucion. Para ellos se publica una
discusion de enfoques y tradeoffs dentro del capitulo.
