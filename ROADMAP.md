# ROADMAP

Estado de avance de `rust-data-structures`, repositorio del camino troncal de
Jeresoft Academy para estructuras de datos en Rust.

No hay fechas limite: este es un proyecto de legado (RFC-0001 §1). Este archivo
orienta el avance, pero no convierte el curso en una carrera por terminar.

## Estado Actual

El repositorio acaba de nacer. La primera meta es crear la fundacion del curso:
identidad, crate Rust, estructura de documentacion, CI y el primer capitulo
publicable.

El checklist detallado vive en
[`docs/superpowers/plans/2026-07-14-rust-data-structures-course.md`](docs/superpowers/plans/2026-07-14-rust-data-structures-course.md).

## Alineacion RFC-0001

- Este repositorio sigue la plantilla de repositorio de RFC-0001 §15.
- Cada capitulo debe cumplir la anatomia de RFC-0001 §14.
- Cada ejercicio debe seguir los niveles de RFC-0001 §17.
- El uso de IA se rige por RFC-0001 §20: la IA acelera, el criterio humano
  decide.

## Fuera De Alcance Por Ahora

- Cursos complementarios o futuros repositorios.
- Algoritmos que pertenecen a `rust-algorithms`, salvo los necesarios para
  explicar una estructura.
- Visualizaciones interactivas hasta que `academy-web` defina el mecanismo de
  ingestion de contenido y playgrounds.
