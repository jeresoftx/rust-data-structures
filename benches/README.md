# Benchmarks

Los benchmarks existen para confrontar teoria y medicion, no para presumir
numeros. Cada benchmark debe estar conectado con una afirmacion de complejidad
del capitulo correspondiente.

Convencion de nombres:

```text
<modulo>_bench.rs
```

Si una medicion contradice la intuicion o la teoria, se documenta como hallazgo
y se explica en el capitulo.
